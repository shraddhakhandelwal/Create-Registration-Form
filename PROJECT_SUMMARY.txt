OPTION 1: Quick Start (H2 Database - No Setup)

     $ mvn clean install

  2. Run with dev profile:
     $ mvn spring-boot:run -Dspring-boot.run.profiles=dev

  3. Access:
     ‚Ä¢ Web: http://localhost:8080
     ‚Ä¢ H2 Console: http://localhost:8080/h2-console

  1. Create MySQL database:
     $ mysql -u root -p
     mysql> CREATE DATABASE registration_db;
     mysql> EXIT;

  2. Update application.properties:
     spring.datasource.username=YOUR_USERNAME
     spring.datasource.password=YOUR_PASSWORD

  3. Build and run:
     $ mvn clean install
     $ mvn spring-boot:run

  4. Access:
     ‚Ä¢ Web: http://localhost:8080
